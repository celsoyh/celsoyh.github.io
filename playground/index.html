atvImg Plugin

Here's the use of the atvImg plugin, made by Drew Wilson.

I used it just to see how it works, but instead of using the same example that he used to show us how his plugin works, I used the Designmodo layers from its tutorial to experiment how it would be if I added more layers, so I could see how it behaves.
